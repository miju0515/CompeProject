# 오늘할일 TODAYTODO



## 프로젝트 설명

'오늘할일 TODAYTODO'는 창직종합설계프로젝트(1)/(2)의 수업에서 만든 프로젝트로 우선순위를 정하기 어려운
사람들에게 우선순위를 추천해주고 사용자가 지정한 우선도와 선호도에 따라 그래프에 표현해주는
할 일 관리 웹서비스입니다.
